# Implementation-and-RTL-Design-of-DMA-Controller-using-Verilog-in-Xilinx-ISE-14.7
## Description
Designed and simulated a Direct Memory Access (DMA) controller using Verilog HDL in Xilinx ISE 14.7 to enable efficient data transfer between memory and peripherals without continuous CPU intervention.

##  Tools & Technologies
- Verilog HDL
- Xilinx ISE 14.7

##  Features
- FSM-based design (IDLE, READ, READ_WAIT, WRITE, DONE)
- Memory interfacing and address generation logic
- Control signals for request, acknowledge, and read/write operations
- Testbench-based functional verification

##  Results
- Reduced CPU workload
- Improved system throughput
- Verified correct timing and data transfer via simulation

##  Applications
- Embedded systems
- High-speed data transfer systems
- SoC and memory controller design
